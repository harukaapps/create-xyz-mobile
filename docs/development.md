# 開発ガイド

## プロジェクト構造

```
react-capacitor2/
├── src/
│   ├── App.jsx          # メインアプリケーションコンテナ
│   ├── MyApp.jsx        # アプリケーション1 (WebGLキャンバス)
│   ├── MyApp2.jsx       # アプリケーション2 (画像処理)
│   ├── assets/          # 静的アセット
│   └── utilities/       # ユーティリティ関数
├── android/             # Androidプロジェクト
├── docs/               # ドキュメント
└── [設定ファイル群]
```

## 主要コンポーネント

### App.jsx
- アプリケーション全体のコンテナ
- アプリケーション切り替え機能の実装
- 状態管理: `currentApp` state

### MyApp.jsx (アプリケーション1)
- WebGL2を使用したキャンバス実装
- タッチイベント処理
- シェーダープログラムの管理

### MyApp2.jsx (アプリケーション2)
- 画像アップロードと処理機能
- 多言語サポート (日本語/英語)
- パラメータ調整UI

## 開発のヒント

1. WebGL開発
- シェーダーコードの変更時は、`vertexSource`と`fragmentSource`を確認
- キャンバスサイズは`devicePixelRatio`に基づいて自動調整

2. UI/UX開発
- モバイルファーストのデザイン
- タッチイベントとマウスイベントの両対応
- レスポンシブデザインの維持

3. パフォーマンス最適化
- WebGLコンテキストの適切な管理
- 不要なレンダリングの防止
- メモリリークの防止

## ビルドとデプロイ

1. 開発ビルド
```bash
npm run dev
```

2. プロダクションビルド
```bash
npm run build
```

3. モバイルビルド
```bash
npx cap sync
```
